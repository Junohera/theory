# Client

## install client

1. [instant-client](https://edelivery.oracle.com/osdc/faces/SoftwareDelivery)
2. search Oracle Database ${VERSION}
3. continue
4. check Oracle Database Client
5. select platform
6. continue
7. download DownLoadManager
8. run DLM
