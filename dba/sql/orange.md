# Orange

## shortcut

- ctrl + -          : 주석처리
- ctrl + shift + -  : 주석해제
- ctrl + enter      : 명령어 실행
- ctrl + u          : 소문자로 변환
- ctrl + shift + u  : 대문자로 변환
- ctrl + s          : 저장
- ctrl + n          : 새접속
- ctrl + t          : 탭추가
- ctrl + o          : 파일불러오기
- ctrl + shift + f  : format
